public void handleLoadCourses Lire un fichier texte contenant des informations sur les cours et les transformer en liste d'objets 'Course'. La méthode filtre les cours par la session spécifiée en argument. Ensuite, elle renvoie la liste des cours pour une session au client en utilisant l'objet 'objectOutputStream'.
@param arg la session pour laquelle on veut récupérer la liste des cours
public void handleRegistration Récupérer l'objet 'RegistrationForm' envoyé par le client en utilisant 'objectInputStream', l'enregistrer dans un fichier texte et renvoyer un message de confirmation au client.
La méthode gère les exceptions si une erreur se produit lors de la lecture de l'objet, l'écriture dans un fichier ou dans le flux de sortie.
