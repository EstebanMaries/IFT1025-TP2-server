public class Server

Représente la relation client avec le serveur.

constructeur Description
Server(int port) Crée un nouveau serveur qui écoute sur le port spécifié. @param port Le port sur lequel le serveur doit écouter. @throws IOException Si une erreur se produit lors de la création du socket du serveur.
